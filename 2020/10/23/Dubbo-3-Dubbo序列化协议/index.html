DubboвҖ”вҖ”3. DubboеәҸеҲ—еҢ–еҚҸи®®

еҸ‘иЎЁдәҺ 2020-10-23 еҲҶзұ»дәҺ Dubbo йҳ…иҜ»ж¬Ўж•°пјҡ

еүҚиЁҖ

  иҝҷзҜҮж–Үз« еҸ–иҮӘиҝҷзҜҮеҚҡе®ўпјҢдҪңдёәж•ҙзҗҶе’Ңж”¶еҪ•пјҢж–№дҫҝеҗҺз»ӯжҹҘйҳ…гҖӮ

  еәҸеҲ—еҢ–пјҢе°ұжҳҜжҠҠж•°жҚ®з»“жһ„жҲ–иҖ…жҳҜдёҖдәӣеҜ№иұЎпјҢиҪ¬жҚўдёәдәҢиҝӣеҲ¶дёІзҡ„иҝҮзЁӢпјҢиҖҢеҸҚеәҸеҲ—еҢ–жҳҜе°ҶеңЁеәҸеҲ—еҢ–иҝҮзЁӢдёӯжүҖз”ҹжҲҗзҡ„дәҢиҝӣеҲ¶дёІиҪ¬жҚўжҲҗж•°жҚ®з»“жһ„жҲ–иҖ…еҜ№иұЎзҡ„иҝҮзЁӢгҖӮ

dubbo ж”ҜжҢҒдёҚеҗҢзҡ„йҖҡдҝЎеҚҸи®®

  • dubbo еҚҸи®®

  й»ҳи®Өе°ұжҳҜиө° dubbo еҚҸи®®пјҢеҚ•дёҖй•ҝиҝһжҺҘпјҢиҝӣиЎҢзҡ„жҳҜ NIO ејӮжӯҘйҖҡдҝЎпјҢеҹәдәҺ hessian дҪңдёәеәҸеҲ—еҢ–еҚҸи®®гҖӮдҪҝз”Ёзҡ„еңәжҷҜжҳҜпјҡдј иҫ“ж•°жҚ®йҮҸе°ҸпјҲжҜҸж¬ЎиҜ·жұӮеңЁ 100kb д»ҘеҶ…пјүпјҢдҪҶжҳҜе№¶еҸ‘йҮҸеҫҲй«ҳгҖӮ

  дёәдәҶиҰҒж”ҜжҢҒй«ҳе№¶еҸ‘еңәжҷҜпјҢдёҖиҲ¬жҳҜжңҚеҠЎжҸҗдҫӣиҖ…е°ұеҮ еҸ°жңәеҷЁпјҢдҪҶжҳҜжңҚеҠЎж¶Ҳиҙ№иҖ…жңүдёҠзҷҫеҸ°пјҢеҸҜиғҪжҜҸеӨ©и°ғз”ЁйҮҸиҫҫеҲ°дёҠдәҝж¬ЎпјҒжӯӨж—¶з”Ёй•ҝиҝһжҺҘжҳҜжңҖеҗҲйҖӮзҡ„пјҢе°ұжҳҜи·ҹжҜҸдёӘжңҚеҠЎж¶Ҳиҙ№иҖ…з»ҙжҢҒдёҖдёӘй•ҝиҝһжҺҘе°ұеҸҜд»ҘпјҢеҸҜиғҪжҖ»е…ұе°ұ 100 дёӘиҝһжҺҘгҖӮз„¶еҗҺеҗҺйқўзӣҙжҺҘеҹәдәҺй•ҝиҝһжҺҘ NIO ејӮжӯҘйҖҡдҝЎпјҢеҸҜд»Ҙж”Ҝж’‘й«ҳе№¶еҸ‘иҜ·жұӮгҖӮ

  й•ҝиҝһжҺҘпјҢйҖҡдҝ—зӮ№иҜҙпјҢе°ұжҳҜе»әз«ӢиҝһжҺҘиҝҮеҗҺеҸҜд»ҘжҢҒз»ӯеҸ‘йҖҒиҜ·жұӮпјҢж— йЎ»еҶҚе»әз«ӢиҝһжҺҘгҖӮ

  иҖҢзҹӯиҝһжҺҘпјҢжҜҸж¬ЎиҰҒеҸ‘йҖҒиҜ·жұӮд№ӢеүҚпјҢйңҖиҰҒе…ҲйҮҚж–°е»әз«ӢдёҖж¬ЎиҝһжҺҘгҖӮ

  • rmi еҚҸи®®

  иө° Java дәҢиҝӣеҲ¶еәҸеҲ—еҢ–пјҢеӨҡдёӘзҹӯиҝһжҺҘпјҢйҖӮеҗҲж¶Ҳиҙ№иҖ…е’ҢжҸҗдҫӣиҖ…ж•°йҮҸе·®дёҚеӨҡзҡ„жғ…еҶөпјҢйҖӮз”ЁдәҺж–Үд»¶зҡ„дј иҫ“пјҢдёҖиҲ¬иҫғе°‘з”ЁгҖӮ

  • hessian еҚҸи®®

  иө° hessian еәҸеҲ—еҢ–еҚҸи®®пјҢеӨҡдёӘзҹӯиҝһжҺҘпјҢйҖӮз”ЁдәҺжҸҗдҫӣиҖ…ж•°йҮҸжҜ”ж¶Ҳиҙ№иҖ…ж•°йҮҸиҝҳеӨҡзҡ„жғ…еҶөпјҢйҖӮз”ЁдәҺж–Үд»¶зҡ„дј иҫ“пјҢдёҖиҲ¬иҫғе°‘з”ЁгҖӮ

  • http еҚҸи®®

  иө°иЎЁеҚ•еәҸеҲ—еҢ–гҖӮ

  • webservice

  иө° SOAP ж–Үжң¬еәҸеҲ—еҢ–гҖӮ

dubbo ж”ҜжҢҒзҡ„еәҸеҲ—еҢ–еҚҸи®®

  dubbo ж”ҜжҢҒ hessionгҖҒJava дәҢиҝӣеҲ¶еәҸеҲ—еҢ–гҖҒjsonгҖҒSOAP ж–Үжң¬еәҸеҲ—еҢ–еӨҡз§ҚеәҸеҲ—еҢ–еҚҸи®®гҖӮдҪҶжҳҜ hessian жҳҜе…¶й»ҳи®Өзҡ„еәҸеҲ—еҢ–еҚҸи®®гҖӮ

иҜҙдёҖдёӢ Hessian зҡ„ж•°жҚ®з»“жһ„

  Hessian зҡ„еҜ№иұЎеәҸеҲ—еҢ–жңәеҲ¶жңү 8 з§ҚеҺҹе§Ӣзұ»еһӢпјҡ

  • еҺҹе§ӢдәҢиҝӣеҲ¶ж•°жҚ®
  • boolean
  • 64-bit dateпјҲ64 дҪҚжҜ«з§’еҖјзҡ„ж—Ҙжңҹпјү
  • 64-bit double
  • 32-bit int
  • 64-bit long
  • null
  • UTF-8 зј–з Ғзҡ„ string

  еҸҰеӨ–иҝҳеҢ…жӢ¬ 3 з§ҚйҖ’еҪ’зұ»еһӢпјҡ

  • list for lists and arrays
  • map for maps and dictionaries
  • object for objects

  иҝҳжңүдёҖз§Қзү№ж®Ҡзҡ„зұ»еһӢпјҡ

  • refпјҡз”ЁжқҘиЎЁзӨәеҜ№е…ұдә«еҜ№иұЎзҡ„еј•з”ЁгҖӮ

дёәд»Җд№Ҳ PB зҡ„ж•ҲзҺҮжҳҜжңҖй«ҳзҡ„пјҹ

  е…¶е®һ PB д№ӢжүҖд»ҘжҖ§иғҪеҰӮжӯӨеҘҪпјҢдё»иҰҒеҫ—зӣҠдәҺдёӨдёӘпјҡз¬¬дёҖпјҢе®ғдҪҝз”Ё proto зј–иҜ‘еҷЁпјҢиҮӘеҠЁиҝӣиЎҢеәҸеҲ—еҢ–е’ҢеҸҚеәҸеҲ—еҢ–пјҢйҖҹеәҰйқһеёёеҝ«пјҢеә”иҜҘжҜ” XML е’Ң JSON еҝ«дёҠдәҶ 20~100 еҖҚпјӣз¬¬дәҢпјҢе®ғзҡ„ж•°жҚ®еҺӢзј©ж•ҲжһңеҘҪпјҢе°ұжҳҜиҜҙе®ғеәҸеҲ—еҢ–еҗҺзҡ„ж•°жҚ®йҮҸдҪ“з§Ҝе°ҸгҖӮеӣ дёәдҪ“з§Ҝе°ҸпјҢдј иҫ“иө·жқҘеёҰе®Ҫе’ҢйҖҹеәҰдёҠдјҡжңүдјҳеҢ–гҖӮ
